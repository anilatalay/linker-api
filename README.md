## Links

| HTTP   | Route          | Header | Body               | Description        |
| ------ | -------------- | ------ | ------------------ | ------------------ |
| `GET`  | /api/links     |        |                    | List all links.    |
| `GET`  | /api/links/:id |        |                    | Get a link.        |
| `POST` | /api/links     |        | {"title":"TITLE" } | Create a new link. |
